## Setup your development

Learn how to get Sonatic up and running

### Step 1: Download necessary dependencies

---

<CodeGroup>
  ```txt python
  pip install @sonatic/sonatic-sdk
  ```

```txt npm
npm install @sonatic/sonatic-sdk
```

</CodeGroup>

### Step 2: Import wrapper package

---

<CodeGroup>
  ```python python
  from @sonatic/sonatic-sdk import SonaticWrapper
  ```

```javascript javascript
import SonaticWrapper from @sonatic/sonatic-sdk
```

</CodeGroup>

### Step 3: Add web navigation abilities with just one line of code

Give access by wrapping your existing client in a single line of code (**only supports OpenAI currently**). All other code remains the same! The wrapper handles integrating and executing desired web automations.

<CodeGroup>
```python python
from openai import OpenAI
oai_client = OpenAI()

client = SonaticWrapper(oai_client)

response = client.chat.completions.create(
messages=[{"role": "system", "message": "Say this is a test" }],
model="gpt-4",
)

print(response)

```javascript javascript
import OpenAI from "openai";

const oai_client = new OpenAI();

const client = SonaticWrapper(oai_client);

const chatCompletion = await client.chat.completions.create({
  messages: [{ role: "user", content: "Say this is a test" }],
  model: "gpt-4",
});

console.log(chatCompletion);
```

</CodeGroup>
